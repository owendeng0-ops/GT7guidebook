# Changelog

本项目使用语义化版本记录主要产品能力变化。V1 保留为原始赛道资料库基线，V1.1 起进入布局级训练助手方向。

## V1.3.0 - 2026-05-16

### 版本定位

遥测实时读取测试版。先验证 PS5 / GT7 UDP 遥测链路，并把实时数据接入主站仪表盘，为后续自动生成训练记录打基础。

### 主要变更

- 新增 `telemetry-agent` 本地测试工具：监听 UDP `33740`，向 PS5 `33739` 发送心跳，解密解析 GT7 telemetry packet。
- 新增 `http://127.0.0.1:8787/health` 体检接口，可查看 PS5 IP、包数、解密数、采样率和错误提示。
- 新增 `ws://127.0.0.1:8787/live` 实时流，主站可订阅 `agent_status` 与 `telemetry_tick`。
- 主站新增“遥测连接”面板，实时显示连接状态、PS5 来源、采样率、车速、挡位、转速、油门和刹车。
- 新增轻量完圈检测：优先读取 `lastLapMs`，当 GT7 模式不提供该字段时使用圈号递增生成 `lap_completed` 候选。
- 主站新增“待归档遥测圈”，完成圈后可先确认，再手动归档到当前 layout 的训练记录。
- 已在真实 PS5 `192.168.3.78` 与 GT7 驾驶场景中验证：约 60Hz 接收，解密错误为 0。

### 注意

- GT7 UDP telemetry 属于社区逆向接口，非官方 API，可能随游戏更新变化。
- 当前版本只做实时读取与展示，自动完圈识别和写入训练记录放到后续迭代。

## V1.2.0 - 2026-05-16

### 版本定位

手动训练闭环版本。围绕“选择今天要练的 layout -> 记录本次练习 -> 复盘差距 -> 下次继续”的使用路径，把原来的单条最佳圈速记录升级为多次练习记录和轻量复盘。

### 主要变更

- 训练仪表盘升级为“今天练什么”：优先展示当前训练、最近一次练习、本周练习次数、复盘信号和下一条推荐。
- 本地训练数据新增 V1.2 存储层 `gt7-track-atlas-training-v1.2`，兼容读取旧版 `gt7-track-atlas-training-v1`。
- 每个官方 layout 支持多次练习 session，记录圈速、车辆、轮胎/辅助设置、失误点、练习感受和创建时间。
- 训练卡片升级为训练工作台：目标区、本次练习记录、三段训练任务、最近 5 次复盘和操作按钮整合在同一模块。
- 个人最佳圈速改为从旧版最佳圈速与 V1.2 sessions 中自动计算，不覆盖官方 GT7 数据。
- 车辆推荐卡片增加训练目标标签，并在用户记录过车辆后标记“上次使用”。
- 赛道索引状态改为参考该赛道内最值得继续练的推荐 layout，而不是只做地点级摘要。

### 数据与体验

- 保持纯静态站结构，继续支持 `file://` 离线打开。
- 不接入 PSN、GT7、账号系统、云同步或后端服务。
- 空数据、旧数据和损坏本地数据会回退到默认记录，不影响页面加载。

## V1.1.3 - 2026-05-14

### Bugfix

- 修复弯角热点定位基准：热点层改为绑定到正方形布局图 surface，而不是整块主图外框，避免宽屏留白导致点位偏移。
- 为 Nürburgring Nordschleife、Endurance II 和 GP 增加静态校准点位，点位按图片像素采样校验，确保落在布局图赛道线上。
- 官方校准点默认展示；用户手工校准后，本地校准点优先，面板会区分“官方校准点”和“本地校准点”。

## V1.1.2 - 2026-05-13

### Bugfix

- 以 Nürburgring 为样板补充 layout 级弯角名称库：GP、Sprint、Nordschleife、Tourist、Endurance、Endurance II 与 24h 会按布局自动使用对应名称顺序。
- 校准模式新增“下一个点位名”和横向名称预览，避免纽北这种长赛道把面板撑爆。
- 新增“自动补名”操作，可把已标定点位按当前布局名称库重新填充名称。

## V1.1.1 - 2026-05-13

### Bugfix

- 停用未校准的自动弯角热点，避免把猜测点位当成真实弯角标注展示。
- 新增本地弯角校准模式：可在布局图上点击记录点位、编辑弯角名、撤销、清空并导出 JSON。
- 弯角点位按 layout ID 保存到 `localStorage`，不覆盖 GT7 官方数据，也不影响其他布局。
- 切换赛道或布局时自动退出校准模式，避免误操作串到其他布局。

## V1.1.0 - 2026-05-13

### 版本定位

进阶玩家训练助手。围绕“选择布局 -> 设定目标圈速 -> 记录最佳圈速 -> 下次继续练”的训练闭环重构。

### 主要变更

- 新增训练仪表盘：进行中布局、已达标布局、最近练习和下一条推荐练习。
- 新增本地训练记录：按官方 layout ID 保存最佳圈速、目标难度、状态、收藏和笔记。
- 新增布局级深链：支持 `#track=Nurburgring&layout=2066d9` 直接回到指定布局。
- 新增多布局对比表：同一赛道内对比长度、弯角、最长直道、个人最佳和训练状态。
- 最佳圈速输入改为固定三段格式：分、秒、毫秒只填数字。
- 赛道索引增加中文对照名，仪表盘赛道名称统一引用中文翻译。
- 布局图增加弯角 hover 热点，真实赛道优先显示公开弯名，原创赛道使用训练弯标注。
- 修复赛道列表筛选隐藏问题，搜索结果不再残留不匹配赛道。

### 数据与体验

- 保持 41 个赛道地点、121 个官方布局基准。
- 保持 GT7 官方数据为主，真实赛道信息用于交叉校验。
- 继续保持静态站结构，支持 `file://` 离线打开，不引入后端。

## V1.0.0 - 2026-05-12

### 版本定位

GT7 赛道资料库。按 World Circuits 大陆分区整理赛道，提供基础检索、布局资料、目标圈速和车辆建议。

### 主要能力

- 41 个 GT7 赛道地点索引。
- 121 个官方布局数据基准。
- 赛道详情页：赛道性格、练习重点、车辆与调校方向。
- 难度圈速指南：新手、进阶、高手三个训练区间。
- 布局级赛道图、车辆缩略图和真实赛道长度校验。
- 搜索、区域筛选和难度筛选。
